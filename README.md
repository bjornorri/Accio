## Installation

### Homebrew

```bash
brew install --cask bjornorri/apps/accio
```

### Direct Download

1. Download the latest release from [Releases](https://github.com/bjornorri/Accio/releases)
2. Open the DMG file and drag Accio to your Applications folder
